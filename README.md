# Python + Tkinter

A simple Login Page application built using Python, Tkinter, and Pillow (PIL).

## Features

- User-friendly Login Interface
- Email and Password Input Fields
- Login Validation
- Success and Error Pop-up Messages
- Custom Application Icon
- Flipkart-inspired UI Theme
- Image Integration using Pillow (PIL)

## Technologies Used

- Python
- Tkinter
- Pillow (PIL)
- PyCharm IDE

## How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/python-tkinter-login-page.git
```

2. Install Pillow

```bash
pip install pillow
```

3. Run the application

```bash
python main.py
```
